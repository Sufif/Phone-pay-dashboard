# 📱 PhonePe Dashboard - Power BI

## 📌 Project Overview

The **PhonePe Dashboard** is an interactive Power BI report designed to analyze digital payment transactions across India. It provides comprehensive insights into transaction volume, transaction value, user engagement, and regional performance through intuitive and interactive visualizations.

2. Open **PhonePay Dashboard.pbix** in Power BI Desktop.

4. Explore the dashboard using interactive filters and slicers.

Or simply use the **Live Dashboard** link above to view the report online.
